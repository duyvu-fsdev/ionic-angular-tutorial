• CustomDatePipe

    • Syntax :<el> ... {{ value | customDate : reverse(boolean) }} ...</el>

    • {{ date }}

    • {{ date | customDate }}

      default - (YYYY/MM/DD - hh:mm:ss)

    • {{ date | customDate : true }}

      reverse : true - (hh:mm:ss - DD/MM/YYYY)
  • GmailGeneratePipe

    • Syntax :<el> ... {{ value | gmailGenerate }} ...</el>

    • {{ email | gmailGenerate }}

  • CustomAsyncPipe

    • Syntax :<el> ... {{ Observable / Subscribable / Promise | customAsync }} ...</el>

    • Observable

    • {{ timeObservable | customAsync }}

  • CustomCurrencyPipe

    • Syntax :<el> ... {{ value | customCurrency : 'currencyCode' }}...</el> ...</el>

    • {{ currencyValue | customCurrency }}

      default VND

    • {{ currencyValue | customCurrency : 'EUR' }}

      currencyCode : EUR

    • {{ currencyValue | customCurrency : currencyCode || 'VND' }}

      currencyCode : {{ currencyCode || 'VND' }}
