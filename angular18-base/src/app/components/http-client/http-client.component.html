    • npm init

    • Name the project and select entry point as

      server.js

    • npm install express sqlite3 cors

    • Create

      server.js

      &

      sqlitedb.js

      files
    • Check code on github
      • Get code server.js Get code sqlitedb.js
    • Finish

      npm run start

    • Api menu list :
      • Get all menus with get method

        http://localhost:8000/api/menu

      • Get menu by id with get method

        http://localhost:8000/api/menu/:id

      • Create (post) new menu with post method

        http://localhost:8000/api/menu

      • Update menu by id with put method

        http://localhost:8000/api/menu/:id

      • Delete menu by id with delete method

        http://localhost:8000/api/menu/:id
